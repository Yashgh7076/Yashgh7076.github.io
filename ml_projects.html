Machine Learning Projects

Predicting on Census Data

This project was part of my Machine Learning course at TU Delft. It was created as a Kaggle competition, seen here Final Assignment IN4320. The goal was to predict if a person would earn 40,000 Euros a year based on census data.

Challenges:

  • Handling missing data entries
  • Dealing with both continuous and categorical features
  • No particular ordering in categorical variables

Approach:

  1. One-hot encoding for categorical features
  2. Estimating distance from prototype examples
  3. Filling in missing values
  4. Using a feed-forward neural network for classification

Data Processing

One-hot Encoding:

Categorical features were converted to one-hot encoded representation to avoid implying an order or importance based on numeric values.

Handling Missing Data:

For continuous features, we used mean imputation. For categorical features, we used the mode (most commonly occurring category).

Classification Model

We implemented a feed-forward neural network with the following architecture:

  • Two hidden layers (150 neurons and 100 neurons)
  • Binary sigmoid activation function for hidden layers
  • Single neuron output layer with binary sigmoid activation
  • Threshold set close to 0.7 for final classification

The overall classification accuracy of the neural network classifier was 83.45%, which was the highest achieved among all classifiers we tried.
